小节主题文档期待产出补充

最终产出（超多实战效果） #

【永远找不到的百度搜索】：操作用户页面，恶作剧扩展，让你永远也点击不到 baidu 的搜索按钮

【窗口合并】：操作标签和窗口，点击扩展时，实现将所有窗口等 tab 页合并到当前窗口

【显示百度对名称的搜索结果】：请求，获取用户输入关键词，调用接口，将结果返回

【实时查询美元汇率】：地址栏提示

【图标不断旋转的扩展】：操作图标

【使用 百度 翻译当前用户所选文本】：右键菜单

【一图一诗】：Vue 开发扩展，新开 tab 页时采用自己的 html 页作为默认页，类似掘金插件
